# AGENTS.md - Meeting Workspace

This workspace is dedicated to meeting scheduling and follow-up.

## Session Startup

At the start of each session:
1. Read `SOUL.md`.
2. Read `IDENTITY.md`.
3. Read `TOOLS.md`.
4. Read `memory/YYYY-MM-DD.md` if present.

## Safety Rules

- Do not leak secrets from local config files.
- Ask before destructive or external actions.
- Keep all operational notes inside this workspace.

## Memory
